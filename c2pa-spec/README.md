# c2pa-spec

Rust types for the [C2PA core specification schemas bundle][schemas-zip].
The types are generated by the [`c2pa-spec-codegen`](../c2pa-spec-codegen)
workspace binary and committed alongside this crate, so downstream builds
don't pull in `cddl`, proc macros, or any network access.

[schemas-zip]: https://spec.c2pa.org/specifications/specifications/2.4/specs/_attachments/C2PA_Schemas.zip

`schemas/` mirrors the upstream C2PA schemas bundle (`cddl/*.cddl`,
`valid_metadata_fields.yml`, `c2pa_urn.abnf`). The Rust sources emitted
from those live under `src/`: `generated.rs` and
`valid_metadata_fields.rs` are both written by `c2pa-spec-codegen` and
should not be edited by hand. Everything else in `src/` is
hand-maintained.

## Excluded CDDL files

Four of the 39 files do not parse with `cddl` 0.10.5 and are skipped
when building the master schema (see [`CDDL_ISSUES.md`](./CDDL_ISSUES.md)):

- `ai-disclosure.cddl`: `1*` occurrence prefix used outside a group/array
- `jsonld.cddl`: `$$group-socket` used in a map-value position
- `regions-of-interest.cddl`: unescaped regex backslashes inside a tstr
- `time-stamp.cddl`: `$$group-socket` used in a map-key position

Symbols from these files that other schemas still reference
(`RegionMap`, `CoseKey`, `CoseSign1`, `CoseSign1Tagged`) are aliased to
`ciborium::Value` stubs in `src/lib.rs` so the crate still compiles.

## Regenerating

```sh
cargo run -p c2pa-spec-codegen
```

Reads the committed CDDL, YAML, and ABNF files under `schemas/` and
writes `src/generated.rs`, `src/valid_metadata_fields.rs`, and
`schemas/cddl/c2pa.cddl`. Pass `--download` to fetch a fresh copy of the
schemas bundle first:

```sh
cargo run -p c2pa-spec-codegen -- --download
```

Point at a different schemas bundle URL with `--url`, or run against a
different crate root with `--spec-dir PATH`. See
[`c2pa-spec-codegen`](../c2pa-spec-codegen/src/main.rs) for the details
of the codegen — identifier conversion, socket/plug rule merging,
literal-to-enum promotion, the `jumbf-uri-type` substitution that swaps
`String` for `jumbf_uri::JumbfUri`, and so on.
